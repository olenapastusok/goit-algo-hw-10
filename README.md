# goit-algo-hw-10

**Exercise02 summary**

**Порівняння результатів:**

1. Аналітичне значення інтеграла ∫ (з межею від 0 до 2) x**2dx = [x**3/3] (з межею від 0 до 2) = 8/3 ~ 2,666667
2. Функція quad з бібліотеки SciPy дає результат з дуже високою точністю (похибка порядку 10**−14), що практично ідентично аналітичному значенню.
3. Метод Монте-Карло дає наближене значення. Його точність напряму залежить від кількості випадкових точок N.


Метод Монте-Карло має похибку, що зменшується пропорційно до 1/ squard(N). Це означає, що для збільшення точності в 10 разів, потрібно збільшити кількість точок у 100 разів. Для одновимірних інтегралів (як у цьому завданні) метод Монте-Карло менш ефективний, ніж класичні методи чисельного інтегрування (адаптивні методи, що використовує quad).
Проте Монте-Карло стає незамінним при обчисленні багатовимірних інтегралів (складні фігури у 3D-просторі та вище), де класичні методи стають занадто складними.

Метод Монте-Карло підтвердив свою працездатність. Отримане значення близьке до теоретичного 2.666, що демонструє закон великих чисел у дії. Хоча він і не є найбільш точним для простих функцій, він є універсальним інструментом для складних систем.
